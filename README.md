# Calculadora de Álcool e Gasolina

Esta é uma calculadora simples que ajuda a decidir entre abastecer com álcool ou gasolina, baseada no preço de ambos os combustíveis.

## Funcionalidades

- Calcula a melhor opção de combustível com base nos preços informados pelo usuário.
- Interface simples e intuitiva.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Estrutura do Projeto

calculadoraAlcoolGasolina/
├── assets/
│   ├── logo.png
│   └── info.png
├── pages/
│   └── calculo.html
├── scripts/
│   └── script.js
├── styles/
│   └── style.css
├── index.html
└── README.md
└── LICENSE.md

- assets/: Contém arquivos de mídia e informações adicionais.
  - logo.png: Logotipo do projeto.
  - info.png: Logotipo de informações.
- pages/: Contém páginas HTML adicionais.
  - calculo.html: Página que descreve como o cálculo é feito.
- scripts/: Contém arquivos JavaScript.
  - script.js: Arquivo JavaScript com a lógica de cálculo.
- styles/: Contém arquivos CSS.
  - style.css: Arquivo CSS com os estilos da calculadora.
- index.html: Página principal do projeto.
- README.md: Arquivo de documentação.
- LICENSE.md: Arquivo de licença.

## Como Usar

- Clone ou faça o download deste repositório.
- Abra o arquivo index.html no seu navegador.
- Insira o preço do álcool e o preço da gasolina nos campos apropriados.
- Clique no botão "Calcular" para ver o resultado.
